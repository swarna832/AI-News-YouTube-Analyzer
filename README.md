# 🤖 AI News & YouTube Sentiment Analyzer

An AI-powered dashboard that analyzes news articles and YouTube data using Natural Language Processing.

## 🚀 Live Demo

(https://swarna-ai-news-analyzer.streamlit.app/)

## ✨ Features

- News sentiment analysis
- YouTube sentiment analysis
- Sentiment visualization
- Word cloud generation
- Interactive dashboard

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib
- NLP
- YouTube API
